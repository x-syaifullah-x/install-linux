### INSTALL PIPEWIRE-JACK
```bash
apt install --no-install-suggests --no-install-recommends pipewire-jack
```
1. pipewire-jack

- **Setup Configuration**
  ```bash
  ln -sf /usr/share/doc/pipewire/examples/ld.so.conf.d/pipewire-jack-*.conf /etc/ld.so.conf.d  && ldconfig
  ```

- **Test Pipewire Jack**

  - **Install jack-tools**
    ```bash
    sudo apt install --no-install-suggests --no-install-recommends jack-tools
    ```

  - **Running jack_simple_client**
    ```bash
    jack_simple_client
    ```
